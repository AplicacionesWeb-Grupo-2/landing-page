# Branch Plan

All branches start from the same base scaffold.

```text
main
└── develop
    ├── feature/header-hero
    ├── feature/app-features
    ├── feature/showcase-why
    ├── feature/overview-signup
    └── feature/footer
```

| Branch | Base scope |
| --- | --- |
| `feature/header-hero` | Header, navigation, hero, and primary CTA |
| `feature/app-features` | Product feature cards and app capability copy |
| `feature/showcase-why` | Product showcase, why section, social proof, testimonials |
| `feature/overview-signup` | Overview rail, signup/contact section, conversion flow |
| `feature/footer` | Footer, social links, legal links, terms page |
